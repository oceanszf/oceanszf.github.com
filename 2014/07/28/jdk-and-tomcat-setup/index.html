JDK_Tomcat 安装

####JDK 安装####
这里版本选择为 jdk-6u33-linux-x64.bin
1.上传安装文件至默认路径

2.对安装文件授予执行权限
chmod +x jdk-6u33-linux-x64.bin

3.执行安装
./jdk-6u33-linux-x64.bin
mv jdk1.6.0_33 /usr/local
ln -s jdk1.6.0_33 java

4.配置环境变量 在/etc/profile后追加即可
JAVA_HOME=/usr/local/java
JRE_HOME=/usr/local/java/jre
CLASSPATH=$JAVA_HOME/lib:$JAVA_HOME/jre/lib
PATH=$JAVA_HOME/bin:$PATH
export PATH CLASSPATH JAVA_HOME

source /etc/profile # 使修改的配置文件生效

5.检测java 是否成功安装
[root@p2ptest ~]# java -version
java version “1.6.0_33”
Java(TM) SE Runtime Environment (build 1.6.0_33-b04)
Java HotSpot(TM) 64-Bit Server VM (build 20.8-b03, mixed mode)

####安装Tomcat####
1.解压文件
tar zxf apache-tomcat-7.0.2.tar.gz -C /usr/local
mv apache-tomcat-7.0.2 tomcat

2.vi /usr/local/tomcat/bin/catalina.sh 添加
JAVA_OPTS=”-Xms1024m -Xmx1024m -Xss1024K -XX:PermSize=128m -XX:MaxPermSize=256m”

配置文件： /usr/local/tomcat/conf/server.xml

日志输出： /usr/local/tomcat/logs/catalina.out

连接串：/usr/local/tomcat/conf/context.xml

    <resource name="jdbc/myoracle" auth="Container"   

    type="javax.sql.DataSource" driverClassName="oracle.jdbc.OracleDriver"   

    url="jdbc:oracle:thin:@127.0.0.1:1521:mysid" username="scott"   

    password="tiger" maxActive="20" maxIdle="10" maxWait=''-1"></resource>  

部分参数解释：

                          name--------------------jdbc/myoracle中myoracle为你要连接的数据库的名字。

                           driverClassName----JDBC驱动类的完整的名称；

                           maxActive---------------同时能够从连接池中被分配的可用实例的最大数

                           maxIdle------------------可以同时闲置在连接池中的连接的最大数；

                           maxWait------------------最大超时时间，以毫秒计；

                           password----------------用户密码；

                           url--------------------------到JDBC的URL连接,最后面的mysid为你安装时数据库实例的名称；

                           username---------------用户名称

                           validationQuery--------用来查询池中空闲的连接。

或是修改这个文件/usr/local/tomcat/webapps/project_name/WEB-INF/classes/conf/jdbc.properties
